The Most Important Thing

大前提：信頼はコピーできない

「見た目」は真似できます。でも「信頼」は真似できません。まなちゃんにしかない“見た目だけでは真似できない財産”を主役にするのがおすすめです。

✓ 真似できる（コピーOK）

  • サイトの色・写真・レイアウト
  • 文章の「型」「並べ方」
  • 予約やLINEの仕組み
≠

真似できない＝まなちゃんの財産

  • 介護で培ったケアと安心感
  • 抜群の傾聴力（話を聞く力）
  • 妹さん（形成外科医）との連携
介

介護のキャリア

有料老人ホーム・看護小規模多機能で、障害のある方を含め長年ケアに従事。体への細やかな配慮と安心感。高齢の方・障害のある方・そのご家族も安心して通える、他店にない入口になります。

傾

抜群の傾聴力

人の話を聞くのが本当に上手。「ただ揉むだけでなく、悩みを聞いて寄り添ってくれる」カウンセリング重視のサロンに直結。指名・常連につながるいちばんの武器です。

医

妹さんが形成外科医

医療の専門家が身近に。医療的な裏付け・監修、むくみ・術後・傷あとケアの相談、将来の連携メニューも。「医療とつながっている安心感」が信頼の核になります。

Decisions

これから決める8項目

01集客 / 知ってもらう★★★

Googleマップに無料登録する

スマホで「中津市 マッサージ」などと地図検索したとき、お店が表示されるようにする無料の設定。

費用 無料手間 小効果まで 数週間
✓ やると、こうなる

近所の人が「中津市 オイルマッサージ」と検索 → 地図の上の方に表示 → 写真・口コミを見て「ここ良さそう」と予約。広告費0円で“今すぐ来たい近所の客”が来店します。

! やらないと、こうなる

検索しても地図に出てこない。上に表示された別の店へ予約が流れます。良いお店なのに「存在を知られないまま」、ご近所客を取りこぼします。

メリット
  • 完全無料で今日から始められる
  • 「近くで今すぐ」の来店意欲が高い客に強い
  • 口コミが信頼の資産として貯まる
デメリット・注意
  • 写真・営業時間の登録／更新に少し手間
  • 悪い口コミもそのまま表示される
  • 上位表示まで数週間かかることも
結論無料で効果が高く、デメリットがほぼない。最優先で実施。はい
02集客 / 知ってもらう★★★

ホットペッパービューティーに掲載する

日本で一番使われている美容予約サイトに有料で掲載し、新規のお客様に見つけてもらう。

費用 月額＋手数料手間 中即効性 高い
✓ やると、こうなる

「中津 オイルマッサージ」で探す新規客が一覧で発見 → 初回クーポンを見て予約・来店。知名度ゼロの開業直後でも“新規客の流れ”が早くできます。

! やらないと、こうなる

一番人が集まる予約サイトに載らず、新規客の最大の入口を逃す。知名度がない開業期は、なかなか予約が埋まらず静かなスタートに。

メリット
  • 新規集客力が圧倒的（最大の入口）
  • 予約・顧客台帳の管理が楽になる
  • 口コミが集まり信頼づくりになる
デメリット・注意
  • 月額＋送客手数料のコスト
  • クーポン値引き競争に巻き込まれやすい
  • サイト内で他店と価格を比較される
結論開業期はコストを払ってでも使う価値あり。慣れたらLINEへ移行。はい寄り
03集客 / また来てもらう★★★

LINE公式アカウントで予約をまとめる

来店客に友だち追加してもらい、次回予約やお知らせを直接届ける“常連づくり”の仕組み。

費用 基本無料〜手間 小〜中リピート 高い
✓ やると、こうなる

来店客が友だち追加 → 後日「そろそろ疲れ取りに来ませんか？」で再来店。手数料ゼロのまま常連に。予約も問い合わせもLINEに一本化できます。

! やらないと、こうなる

一度来たお客様と連絡手段がなく、そのまま忘れられて再来なし。毎回ゼロから新規集客が必要になり、ずっと集客に追われる経営に。

メリット
  • 手数料ゼロでお客様と直接つながれる
  • リピート・常連化にもっとも強い
  • 予約・問い合わせを1か所に集約
デメリット・注意
  • 友だちを増やす工夫（店内告知等）が必要
  • 配信しすぎるとブロックされる
  • 最初の設定・メニュー作りがやや手間
結論集客の“受け皿”として必須。最初に用意したい仕組み。はい
04集客 / 雰囲気を伝える★★★

Instagramを「1つ」に絞って続ける

個室や施術の雰囲気・お客様の感想を写真で発信し、来店前に“安心して選んでもらう”。

費用 無料手間 中（継続が鍵）即効性 ゆっくり
✓ やると、こうなる

落ち着いた個室・施術の様子・お客様の感想を投稿 → 「気持ちよさそう」と雰囲気で選ばれ、プロフィールのリンクから予約。世界観でファンになってもらえます。

! やらないと、こうなる

来店前に雰囲気が分からず不安で選ばれにくい。「どんな所か分からないから他の店に」と比較で負けやすくなります。

メリット
  • 無料でお店の世界観・安心感を伝えられる
  • 写真映えする業種と相性が抜群
  • ファン・指名・紹介につながる
デメリット・注意
  • 毎日投稿は不要（疲れて世界観がぶれる）
  • 2つ以上のSNS同時運用は続かない
  • すぐには成果が出ない（じわじわ型）
結論やる価値は高いが「1アカウントに集中」が鉄則。はい
05お金 / 来てもらう★★★

料金をサイトにはっきり載せる

初回・通常メニューの値段をサイト上で明記し、初めての人が安心して予約できるようにする。

費用 0円手間 小予約率効果 高い
✓ やると、こうなる

「初回◯◯円・通常◯◯円」と一目で分かり、安心して予約ボタンを押せます。「いくらか聞く」手間がなく、問い合わせ対応の負担も減ります。

! やらないと、こうなる

「いくらか分からない」と不安で離脱。わざわざ問い合わせる人は少なく、見込み客が静かに去る。新店ほどこの取りこぼしが痛い。

メリット
  • 初めての人の不安を消し、予約率が上がる
  • 価格に納得した人だけが来てミスマッチが減る
  • 問い合わせ・電話対応の手間が減る
デメリット・注意
  • 他店と価格を比較されやすくなる
  • 安売りに見せない見せ方の工夫が要る
  • 価格変更のたびに更新が必要
結論隠すより載せる。新店の予約率に直結する。
※中津市の相場目安：初回3,500〜4,500円／通常60分5,000円前後・90分8,000〜9,000円はい
06お金 / 来店〜継続★★★

初回お試し価格 ＋ 回数券・継続プラン

「お試しで来てもらう入口」と「お得に通い続けてもらう継続」をセットで設計し、売上を安定させる。

✓ やると、こうなる

手頃な初回お試しで体験 →「良かった」→ 回数券でお得に継続 → 月1〜2回の常連に。売上が読めるようになり経営が安定します。

! やらないと、こうなる

通常価格だけだと初回ハードルが高く来店が少ない。来ても1回きりで、毎回ゼロから集客。売上が月ごとに乱高下します。

メリット
  • A 初回お試し：来店ハードルが下がり最初の1人を呼べる
  • 体験してもらえば良さが伝わる／口コミのきっかけに
  • B 回数券：前払いで売上・資金繰りが安定／常連化しやすい
デメリット・注意
  • 安すぎると原価割れ。下げ過ぎ注意
  • 「お試しだけ」の客も一定数いる→継続導線が必須
  • 押し売り感はNG／有効期限・残回数の管理が必要
結論入口（お試し）と継続（回数券）はセットで用意する。“安さ”でなく“ケアの価値”で選ばれることを大切に。両方はい
07仕組み / 予約★★★

スマホで完結するネット予約を入れる

お客様がスマホから24時間予約でき、できれば事前決済もできる予約システムを導入する。

費用 無料〜月数千円手間 中（初期設定）取りこぼし防止 高い
✓ やると、こうなる

深夜でもスマホで空き時間を見て予約完了。事前決済つきなら無断キャンセルも激減。施術中で電話に出られなくても予約は自動で入り続けます。

! やらないと、こうなる

電話やDMで予約調整 → 営業中は出られず取りこぼし。ダブルブッキングや無断キャンセルも起きやすく、一人営業ほど負担と損失が増えます。

メリット
  • 24時間、自動で予約が入る
  • 取りこぼし・二重予約を防げる
  • 事前決済でキャンセル損失を減らせる
デメリット・注意
  • 最初の設定に少し手間
  • 月額がかかるサービスもある
  • PCが苦手でも“簡単なもの”を選ぶこと
結論簡単に扱えるものを選んで導入。電話だけは避ける。LINE連携できるタイプが便利。はい
08仕組み / 決済★★★

支払い方法をそろえる（現金＋キャッシュレス）

現金でもカードでもスマホ決済でも払えるように。Airペイ等の「1台でまとめて」端末や、予約時のオンライン事前決済も検討。

手数料 売上の数%手間 中（申込・設定）便利さ 高い
✓ やると、こうなる

カード・PayPay・交通系でサッと支払い。会計がスムーズで、高めのコースやオプションも選ばれやすく。事前決済を使えば無断キャンセルも減ります。

! やらないと、こうなる

現金のみだと「カード使えないの？」とがっかりされ、ATMに行く手間で取りこぼし。現金を持ち歩かない人も増え、高めメニューをためらわれがちに。

選択肢
  • 現金：手数料ゼロ／その場で手元に（0円）
  • Airペイ等：クレカ・電子マネー・QRを1台で（端末実質0円〜＋手数料約3%）
  • オンライン事前決済：無断キャンセル激減・当日会計不要
注意点
  • 現金：釣銭・記帳の手間／持ち合わせ無い客を逃す
  • キャッシュレス：決済手数料・入金サイクルの確認
  • 事前決済：返金・変更ルールと予約連携が必要
結論現金＋キャッシュレス端末は両方用意。事前決済も予約と合わせて導入。タブレット1台で完結する端末を選べばOK。両方はい

この形で作るのがおすすめ

迷ったらおすすめ度の高いものから。★★★＝まず必ず入れる／余裕が出たら追加、の順番です。

★★★ まず必ず入れる（無料〜低コストで効果大）

  • Googleマップ登録 — 地図で見つけてもらう
  • LINE公式 — 常連・再来の受け皿
  • 料金＋初回お試しを載せた1枚サイト
  • 回数券・継続プラン — 売上の安定
  • ネット予約（24時間・取りこぼし防止）
  • キャッシュレス決済（Airペイ等・事前決済）

できれば追加（余力が出たら）

  • Instagram — 1つに集中
  • ホットペッパー — 開業期だけのブースター

サイトの主役にすること

  • 介護のケア・傾聴力・妹さん（形成外科医）連携
  • 料金 ＋ 初回お試し ＋ お客様の声

まずはこの形で作るのがおすすめです。とくにまなちゃんの「聞く力（傾聴）」と「介護で培った安心感」がいちばんの武器。 これがしっかり伝わるサイトにすれば、見た目以上に信頼される強いお店になります。

Things To Avoid

これだけは避けたい

× 人のドーパミンを煽る集客

「今だけ・限定」で不安や射幸心を煽る集客は、ケアの世界観と合わず、その場限りの客しか来ません。

× 「安さ」を売りにする

安さを看板にすると価格で選ばれ、価格でしか続きません。傾聴とケアの価値で選ばれるお店に。

× 数を追う継続投稿

フォロワー稼ぎの量産・毎日投稿に追われると疲れ、世界観もぶれます。質を優先し続けられる範囲で。

× 見た目だけ真似る

強み（介護・傾聴・医療連携）を出さず高級風にすると、かえって安っぽく見えます。
